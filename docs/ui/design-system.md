# Design System

## Goal

Create a professional legal-tech SaaS application similar to:

* Clio
* Rocket Lawyer
* Stripe Dashboard
* Notion
* OpenAI

The application must feel trustworthy, modern, and easy to use.

---

## Principles

### Clarity

Users should always know:

* Where they are
* What information is required
* What happens next

### Simplicity

Avoid:

* Long forms
* Large walls of text
* Complex navigation

Prefer:

* Step-by-step workflows
* Guided conversations
* Progressive disclosure

### Consistency

Use consistent:

* Colors
* Typography
* Spacing
* Buttons
* Form controls

---

## Layout Rules

Desktop:

* Left navigation
* Main content area
* Right contextual panel

Mobile:

* Single-column layout
* Bottom action buttons
* Collapsible summary panels

---

## Cards

All information should be grouped into cards.

Card Types:

* Information Card
* Warning Card
* Success Card
* Document Card
* Workflow Card

---

## Typography

Page Title

* Large
* Bold

Section Heading

* Medium
* Bold

Body Text

* Comfortable reading size

---

## User Trust

Legal software must prioritize:

* Professional appearance
* Clear language
* Visible progress
* Data safety indicators
