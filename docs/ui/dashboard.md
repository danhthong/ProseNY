# Dashboard UI

## Purpose

Provide users with a clear overview of their case.

---

## Layout

Header

Case Progress

Recent Activity

Next Required Actions

Document Status

AI Assistant

## Widgets

### Case Progress

Progress Bar

Completion %

Current Stage

---

### Next Steps

Display:

- Tasks
- Deadlines
- Required actions

---

### Documents

Show:

- Uploaded
- Missing
- Pending Review

---

### AI Assistant Widget

Quick questions

Resume workflow

Ask legal questions

## Empty States

If no case exists:

Show:

Start New Case

with a prominent call to action.

## Mobile

Widgets should stack vertically.

Most important information first:

1. Progress
2. Next Steps
3. Documents
4. AI Assistant
