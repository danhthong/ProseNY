# Intake Flow UI

## Purpose

Collect information from users without overwhelming them.

Never display a 50-question form.

Use guided intake.

---

## Layout

Header

Progress Stepper

Main Content

Current Question

Helpful Explanation

Chat or Form Input

Footer Actions

Back
Save Draft
Continue

---

## Completion Tracking

Always display:

* Completion percentage
* Completed sections
* Missing sections

Example

20% Complete

✓ Personal Information

✓ Marriage Information

○ Children

○ Property

○ Support

○ Documents

---

## Missing Information

Display a dedicated panel:

Missing Information

* Child date of birth
* Marriage date
* Income information

---

## Save Progress

Users should be able to:

* Leave
* Return later
* Continue from last step
